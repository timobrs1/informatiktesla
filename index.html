TESLA

Tesla ist ein börsennotierter US-amerikanischer Autohersteller, der größtenteils Elektroautos produziert. Die Firma wurde im Jahr 2003 von Martin Eberhard und Marc Tarpenning gegründet. Seit 2008 ist Elon Musk Aufsichtsratsvorsitzender und ist heutzutage die prägende Figur des Unternehmens. Inzwischen beschäftigt Tesla weltweit rund 70.000 Mitarbeiter in über 300 Stores. Der Hauptsitz liegt in Austin. Fünf sogenannte „Gigafactorys“ produzieren einen Großteil der Autos.

Model Y

Reichweite maximal 533km
Beschleunigung von 0-100km/h 3,7 Sek
Höchstegeschwindigkeit 250km/h
Leistung 1020 PS

Das Tesla Model Y ist ein batterieelektrisch angetriebener SUV-Crossover des US-amerikanischen Herstellers Tesla. Seit zwei Jahren wird das Auto produziert. Es ist das erste Modell von Tesla, das über ein Wärmepumpensystem zum Heizen des Fahrzeugs verfügt. Das Model Y kann als etwas größerer und technisch etwas ausgereifter Nachfolger des Model 3 gesehen werden.

Model X

Reichweite maximal 536km
Beschleunigung von 0-100km/h 2,6 Sek
Höchstegeschwindigkeit 262km/h
Leistung 1020 PS

Das Tesla Model X von Tesla ist ein allradgetriebener Oberklasse-Pkw mit elektrischem Antrieb. Der „SUV“ wird seit 2015 in Serie hergestellt. Auffällig sind besonders die Flügeltüren. Die zahlreichen Fahrerassistenzsysteme nennt Tesla selbst „Autopilot“. So kann das Auto voll-automatisch fahren. Das Auto gilt als größere aber technisch etwa gleich starke Version des Model S.

Model S

Reichweite maximal 637km
Beschleunigung von 0-100km/h 2,1 Sek
Höchstegeschwindigkeit 322km/h
Leistung 1020 PS

Der Tesla Model S ist ein batterieelektrisch angetriebenes Automodell. Die Oberklasse Limousine wird seit 2012 hergestellt. Jedes Auto ist mit einem Internetzugang ausgestattet, über dem die betriebseigenen Updates eingespielt werden. Zudem verfügt der Model S über die „Autopilot“ genannten Fahrassistenzfunktionen. Damit könne man laut Angabe Teslas vollautonom fahren. Die moderne Technik wie etwa ein 17-Zoll-Touchscreen-Display sorgten mehrfach für Platz 1 der meistverkauften Elektroautos der Welt.
